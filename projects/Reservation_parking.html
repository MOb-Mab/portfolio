Condominium Parking Reservation Application

Project Overview

This project was a prototype mobile application for reserving condominium parking spaces. I designed the user interface in Figma and developed the backend using Firebase and Node.js. The system included real-time parking availability, advance booking, and a straightforward user flow to make the process simple for residents.
