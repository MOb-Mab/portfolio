AR Development Project for Web Application

Project Overview

This project explored integrating augmented reality into a web-based platform to enhance user interaction and engagement. The aim was to deliver an AR experience directly through a browser without the need for additional applications. I handled the website front-end development using HTML, CSS, JavaScript, and React, creating a responsive and intuitive interface.
